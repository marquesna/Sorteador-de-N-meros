Sorteador de Números 🎲
Um pequeno e simpático sorteador de números feito com HTML, CSS e JavaScript — perfeito para decisões rápidas, sorteios de sala de aula ou diversão entre amigos.

✨ O que faz
Gera números aleatórios dentro de um intervalo definido pelo usuário.
Interface leve e responsiva.
Fácil de personalizar e integrar em qualquer página.

🛠️ Tecnologias
HTML
CSS
JavaScript (Vanilla)

▶️ Como usar

Abra o arquivo index.html no navegador (duplo clique).
Ou rode um servidor local rápido:
python -m http.server 8000
Abra http://localhost:8000 no navegador.
Defina o intervalo, clique em “Sortear” e divirta-se!

✍️ Personalizar
Ajuste estilos em styles/ (arquivos CSS).
Lógica e comportamento em scripts.js — mexa nas funções se quiser alterar efeitos ou animações.
